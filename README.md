# GOVERN

A browser-based United States political simulator built as a single-page HTML, CSS and JavaScript project.

## Live site

Add GitHub Pages link here.

## Current features

- Player setup screen
- Party and difficulty selection
- Approval tracking
- Economy indicators
- Cabinet system
- Congress panel
- Budget panel
- World map and foreign relations
- Command centre interface
- Weekly events and decision log

## How to run

Open `index.html` in a browser, or use the GitHub Pages deployment.
